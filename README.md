# SONIC-P-06-MOBILE BY LOWFRIEND
ENG:
SONIC P-06 MOBILE - Unofficial Android port of the fan game by ChaosX.

-Performance Optimization: Adapted the game for mobile devices to ensure stable performance and FPS.

-Graphics & Lighting: Adjusted rendering, graphics, and lighting to work correctly on mobile platforms.

-Input System: Developed and implemented custom touch controls (On-screen UI) from scratch.

RUS:
SONIC P-06 MOBILE - Неофициальный Android-порт фан-игры от ChaosX.

-Оптимизация производительности: Адаптация игры под мобильные устройства (настройка шейдеров, физики и стабильного FPS).

-Работа с рендером: Перенастройка графики и освещения для +-корректного отображения на смартфонах.
